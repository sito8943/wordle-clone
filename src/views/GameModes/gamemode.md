## Zen

- Player never loses
- Infinity rows (for code always add two more at then when the player gets to the end)
- No score influenced
- Infinity green hints
- Accept any letter combination

ruta: /zen

## Classic

_Actual game mode_

- 6 rows and 5 letters

ruta: /clasico

## Lightning

_Actual game mode when insane difficulty_

- 6 rows and 5 letters
- 60 seconds to complete

ruta: /relampago

## Daily

- 6 rows and N letters (Where N is the length of the word of the day)

ruta: /palabra-diaria

For Classic and Lightning

Player can decide to choose

- Can use hints (1 yellow), if not gain extra point when winning
- Can write dictionary words, if yes every dictionary word give extra multiplication factor just as current normal difficulty, if not just be like current hard difficulty

The game stills having one score
